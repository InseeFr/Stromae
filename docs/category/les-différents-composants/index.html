Les différents composants

Cette partie décrit les différents composants proposés dans Stromae DSFR (et qui peuvent être décrits grâce aux outils Pogues, Eno et Lunatic

📄️ Champs texte

Le composant Input permet d'afficher un simple champ texte, caractérisé par une longueur maximale du texte.

📄️ Champ "Nombre"

Le composant InputNumber permet d'afficher un champ numérique.

📄️ Les champs Date

Le composant Datepicker permet à l'utilisateur de sélectionner une date.

📄️ Les durées

📄️ La recherche sur liste

Le composant Suggester permet d'avoir un menu déroulant avec une option de recherche.

📄️ Les liens deux à deux

Le composant lien 2 à 2 permet de définir des relations bidirectionnelles entre les données, principalement les liens familiaux dans les enquêtes ménages de l'Insee.

📄️ Question à choix unique

Le composant Radio Permet de sélectionner une valeur parmi une liste d'options (questions à choix unique).

📄️ Les questions à choix multiples

La question à choix multiples est représentée par un composant CheckboxGroup.

📄️ Les tableaux fixes

Le composant Table est un composant permettant d'afficher un tableau d'information à l'utilisateur.
