Données de l'unités enquêtées

📄️ Contrôles

Les contrôles permettent de définir des règles de validation dans le questionnaire (sous forme d'expression VTL).

Télécharger la preuve de dépôt

📄️ Navigation et "fil d'Ariane"

📄️ Capture de paradonnées
