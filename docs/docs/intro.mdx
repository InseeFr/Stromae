# Introduction

Bienvenue sur la documentation de Stromae DSFR.


:::warning[En cours de construction]
Stromae-DSFR est un orchestrateur qui va être utilisé à partir de l'automne 2024 pour les collectes web des enquêtes auprès des ménages et des entreprises, par l'Insee et certains des services statiques ministériels. Il remplace notammant [Stromae-V2](https://github.com/InseeFr/Stromae/tree/v2-master).

Pourquoi DSFR ? Simplement parce qu'il se base sur le [Système de Design de l'État](https://www.systeme-de-design.gouv.fr/), qui regroupe un ensemble de composants réutilisables, répondant à des standards, pouvant être assemblés pour créer des sites Internet accessibles et ergonomiques.

:::
