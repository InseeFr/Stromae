# Introduction

Bienvenue sur la documentation de Stromae V3


:::warning[En cours de construction]

StromaeV3 a été développé à partir des besoins du recensement de la population en reprenant la couverture fonctionnelle initiale de [Stromae-V2](https://github.com/InseeFr/Stromae/tree/v2-master). Il s'agit d'une refonte de l'application initale et reprenant le DSFR.

Cependant, il n'existe pas de documentation reprenant la couverture fonctionnelle de cette application.
Cette documentation vise à décrire l'existant, le périmètre fonctionnelle afin d'en reprendre pleinement le contrôle de ses fonctionnalités.
:::
